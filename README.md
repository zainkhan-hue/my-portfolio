# 🏀💻 Personal Portfolio Website

A cyberpunk-themed personal portfolio for an athlete, student, and tech enthusiast.

---

## 📁 File Structure

```
my-portfolio/
├── index.html          ← Main HTML file (open this in browser)
├── css/
│   ├── style.css       ← All styles, layout, colors, components
│   └── animations.css  ← Keyframe animations & scroll reveal classes
├── js/
│   ├── main.js         ← Navigation, scroll effects, skill bars, contact form
│   ├── game.js         ← Basketball mini-game (HTML5 Canvas)
│   └── animations.js   ← Extra visual effects, cursor trail, glitch
└── assets/             ← Place your profile photo and any images here
```

---

## 🚀 How to Open in VS Code

1. Open **VS Code**
2. Go to `File → Open Folder` and select the `my-portfolio` folder
3. Right-click `index.html` → **Open with Live Server** (install the Live Server extension if needed)
4. Or simply double-click `index.html` to open it directly in your browser

---

## ✏️ How to Customize

| What to change | Where to find it |
|---|---|
| Your name | `index.html` — search for `YOUR NAME` |
| Your bio | `index.html` — `#about` section |
| Your photo | Replace the placeholder in `#about` with an `<img>` tag |
| Skill percentages | `index.html` — `data-width` attributes in `#skills` |
| Experience entries | `index.html` — `#experience` section |
| Projects | `index.html` — `#projects` section |
| Contact info | `index.html` — `#contact` section |
| Colors | `css/style.css` — `:root` CSS variables at the top |

## 🎮 Mini-Game Instructions

- Click **START GAME** to begin
- **Move your mouse** to aim at the hoop
- **Hold click** (or hold SPACE) to build power, **release** to shoot
- Score as many baskets as you can in **10 shots**
- Streak bonuses: 2 in a row = +2pts, 3+ in a row = +3pts
- High score is saved in your browser
